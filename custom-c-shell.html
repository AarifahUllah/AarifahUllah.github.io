As part of my Systems Programming class, I spent 6 months making a shell that can interpret and execute Linux commands. The main programming language used was C. The shell parsed user commands for errors, tokenized each word, and separated by key words, like semi-colons and pipelines.

Supported functionality included file redirection, changing directories, sleep() commands, and supporting multiple processes.

My implementation went above and beyond what was expected and required, earning extra credit for providing more features like job control as well.
