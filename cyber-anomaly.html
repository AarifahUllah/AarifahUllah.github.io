I study graph neural network (GNN) applications a Research Assistant in GraphLab under Prof. Howie Huang. One application being cybersecurity-related anomaly detection. A network structure is similar to a graph's nodes and edges, so information can be captured in a GNN quite nicely.

Another related GNN application is using it to improve the electric grid infrastructure with power flow optimization and predicting how much power renewable energy resources will output at a given time. The current grid is inefficient so GNNs are capable to finding inefficiencies.
